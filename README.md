# practice-cpp

A collection of C++ programming exercises and solutions for learning and practicing core language features.

## Overview

This repository contains **28 tasks** focused on:
- Basic input/output and control structures
- Arrays and matrices
- Strings manipulation
- Functions and algorithms
- Numerical computations

Each task is organized in its own directory (`task-01` to `task-28`) with:
- `main.cpp` — solution code
- `CMakeLists.txt` — build configuration

## How to Build and Run

```bash
# Build a specific task (example: task-01)
cd task-01
mkdir build && cd build
cmake ..
make
./task-01
```
